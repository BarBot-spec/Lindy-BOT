## Clone this project

```bash
> git clone https://github.com/BarBot-spec/Lindy-BOT
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd Lindy-BOT
> bash install.sh
```

### Usage
```bash
> npm start
```
